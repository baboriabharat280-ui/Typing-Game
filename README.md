# ⌨️ Typing Game

A responsive **Typing Speed Test** web application built using **HTML, CSS, and JavaScript**. The application measures a user's typing speed and accuracy by displaying random text, tracking the time taken, and calculating the final score in real time.

## ✨ Features

* ⏱️ Real-time typing timer
* 🎯 Live score tracking
* ✅ Instant text validation
* 📝 Random typing prompts
* 📱 Responsive and user-friendly interface
* ⚡ Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How to Use

1. Open the application in your browser.
2. Click the **Start** button.
3. Type the displayed text as accurately and quickly as possible.
4. View your completion time and final score.
